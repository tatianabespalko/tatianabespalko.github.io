Консоль (командна строка або термінал) - це ще один спосіб взаємодії з комп'ютером.

Для різних операційних систем початок роботи з терміналом відрізняється:

Windows: Відкрийте меню "Пуск" і в командному рядку виберіть команду execute та введіть cmd.

Mac: Відкрийте Spotlight, тип термінал і відкрийте програму.

Linux: Знаходиться в головному меню або в розділі аксесуари.

Концепція Використання Приклади Опис
Зміна каталогу cd directory

cd my_app

cd my_app/app/controllers

 Змінює каталог на вказаний.
Список вмісту каталогу

ls directory

Windows: dir directory

ls

ls my_app

 Показує весь вміст (файли та папки) каталогу. Якщо не вказано жодного каталогу, відображається вміст поточного каталогу.
Каталог, в якому Ви зараз знаходитесь pwd pwd Показує повний шлях до каталогу, в якому Ви знаходитесь. Наприклад, /home/tobi/railsgirls. Примітка: якщо ім'я файлу або каталогу починається з слешу (/) - це абсолютне або повне ім'я файлу. Якщо слеша немає (/) - ім'я файлу є відносно поточного робочого каталогу.
Створити новий каталог mkdir name

mkdir rails

mkdir fun

 Створює каталог з ім'ям у папці, в якій Ви перебуваєте.
Видалити файл

rm file

Windows: del file

rm foo

rm index.html

rm pictures/old_picture.jpg

Видаляє вказаний файл. Будьте обережні, щоб не видалити необхідні Вам файли/p>

Можна вказати назву файлу каталогу, в якому Ви перебуваєте. Також можна вказати шлях, як у третьому прикладі (видаляється файл з папки зображень).
Видалити каталог

rm -r folder

Windows: rd folder

rm -r stuff_i_dont_need

rm -r stuff_i_dont_need/

rm -r old_application

Видаляє вказану папку та весь її вміст.
